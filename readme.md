### 并查集

### LCA

**模拟**

从一个节点向上找，并且标记路径。从另一个节点也向上走，碰到标记，就是最近公共祖先

**Tarjan**

在深度遍历的同时，用并查集维护节点之间的关系，当访问到某一个查询点时，如果另一个点已经被访问过了，那么查找他所在并查集的顶点就是最近公共祖先。适合多组查询的时候。
